# 📍 Embeddings Playground

An interactive explainer of what an embedding is, from a single audio clip to almost 25,000 tracks. Each track is encoded as a CLAP audio embedding, reduced to 2D/3D with UMAP, and plotted as a point colored by genre. Click any point to stream the track, switch between 2D and 3D, and filter by genre.

## Setup

```bash
npm install
npm run dev
```

## Data

The data files in `public/` are generated by the [embeddings-data-preprocessing](https://github.com/catherinepereira/embeddings-data-preprocessing) pipeline: CLAP encodes each track to 512D, UMAP reduces to 2D and 3D.

- `viz.json`: per-track 2D/3D coordinates, genre, and metadata
- `vectors.bin` + `vectors.meta.json`: the 512D CLAP vectors (float16) and their id index, loaded on demand for the embedding strips
- `genre_vectors.json`: the mean embedding per genre
